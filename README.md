# Minazuki (ja)
シンプルな製品ライセンス認証システム「Minazuki」のサーバーサイド実装です。

このアプリケーションは、商用アプリケーションのシンプルなライセンス認証サーバとして利用することを想定して開発されています。もちろんその他の用途でも利用が可能です。
